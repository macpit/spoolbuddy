# SpoolBuddy ESP32 Display UI Design

> Bambu Lab-inspired UI for the 4.3" touch display (800x480)

---

## Design Philosophy

Match Bambu Lab's aesthetic:
- **Dark theme** - Easy on eyes, looks premium
- **Clean layout** - Plenty of whitespace, clear hierarchy
- **Rounded elements** - Cards, buttons, progress bars
- **Accent colors** - Cyan/teal highlights on dark background
- **Status-focused** - Key info visible at a glance

---

## Color Palette

```
Background:
  Primary:    #1A1A1A (near black)
  Card:       #2D2D2D (dark gray)
  Elevated:   #3D3D3D (lighter gray)

Text:
  Primary:    #FFFFFF (white)
  Secondary:  #B0B0B0 (gray)
  Muted:      #707070 (dark gray)

Accent:
  Primary:    #00ADB5 (cyan/teal - Bambu style)
  Success:    #4CAF50 (green)
  Warning:    #FFC107 (amber)
  Error:      #F44336 (red)

Filament Colors:
  (Dynamic based on spool RGBA)
```

---

## Typography

```
Font: Roboto or Inter (convert to LVGL format)

Sizes:
  Title:      28px bold
  Heading:    20px semibold
  Body:       16px regular
  Caption:    12px regular
  Label:      14px medium
```

## Animation Guidelines

### NFC Scan Animation
- Concentric circles pulsing outward
- Duration: 1.5s loop
- Easing: ease-out

### Weight Update
- Number counter animation
- Duration: 300ms
- Easing: ease-out

### Screen Transitions
- Slide left/right for navigation
- Fade for modals
- Duration: 200ms

### Button Press
- Scale down to 95%
- Duration: 100ms
- Background darken 10%

---

## Touch Targets

- Minimum button size: 48x48 pixels
- Recommended: 64x64 pixels for primary actions
- Spacing between touch targets: 8px minimum

---

## Responsive Considerations

The 800x480 display is fixed, but UI should:
- Use relative positioning where possible
- Scale icons based on content area
- Maintain consistent padding (16px standard)
- Keep important info in center/upper portion (viewing angle)
